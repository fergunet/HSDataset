# HSDataset: Hearthstone Decks Dataset

Dataset used in the paper "Antonio M. Mora, Alberto Tonda, Antonio Fernández-Ares, Pablo García-Sánchez: Looking for archetypes: Applying game data mining to hearthstone decks. Entertain. Comput. 43: 100498 (2022)".

Dataset is available in https://zenodo.org/records/10198504
